# AI Resume Analyzer & Interview Copilot

A full-stack AI-powered platform that analyzes resumes, optimizes content for ATS, and simulates real-time mock interviews using LLMs.

## 🚀 Features

- 📄 Resume Analysis – Evaluate resume-job alignment using AI
- ✍️ Bullet Rewriting – Generate impactful, ATS-friendly resume points
- 🎯 Skill Gap Analysis – Identify missing skills for target roles
- 💼 Job Matching – Suggest best-fit roles based on skills
- 🤖 Interview Copilot – ChatGPT-style mock interview with real-time feedback

## 🛠 Tech Stack

- Frontend: React
- Backend: Node.js, Express
- APIs: REST APIs
- AI: LLMs (OpenAI API)

## 📌 Highlights

- Processed 600+ resumes for analysis and optimization  
- Improved resume quality and ATS compatibility by 30%+  
- Built real-time conversational interface for mock interviews
